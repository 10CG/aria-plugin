# State Scanner - 迁移说明 (v1.0 → v2.0)

> 从 State Scanner v1.0 升级到 v2.0 的变更说明

---

## 变更点

| v1.0 | v2.0 |
|------|------|
| 仅状态扫描 | 状态 + 推荐 + 确认 |
| 方向选择 (A/B/C/D) | 工作流推荐 + 自定义 |
| 输出报告后结束 | 输出到 workflow-runner |

---

## 向后兼容

- 原有的状态扫描功能完全保留
- "只看不执行" 选项保持可用
- UPM 解析逻辑不变
